# Portfolio slike

Ovdje dodajte slike za portfolio projekte.

Struktura:
- `primjer-1/cover.webp` - slika za prvi projekat
- `primjer-2/cover.webp` - slika za drugi projekat
- `primjer-3/cover.webp` - slika za treći projekat

Za nove projekte:
1. Kreirajte folder sa slug-om projekta (npr. `naziv-projekta/`)
2. Dodajte `cover.webp` sliku u taj folder
3. Ažurirajte `data/projects.ts` sa putanjom do slike

Preporučeno:
- Format: WebP (najbolje performanse)
- Dimenzije: 1200x800px ili slično
- Optimizacija: Kompresovane slike za brže učitavanje
